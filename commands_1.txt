dotnet new console  - создаем консоль для работы 
dotnet run  - запускаем программу для работы 
Console.Readline() - вводит в переменную то что набрал в консоле.
Console.WriteLine() - выводит на экран  каждый раз на новую строку 
Console.Write() - выводит на экран в одну строку
init  - целые числа и при делени будет только целое число 
double  - вещественное число, при делении будет остаток в виде десятичной дроби.
new Random().Next(min, max) метод принимает случайное число  мин и макс 
ToLower() принимат буквы и делает их  маленькими 
int a = Convert.ToInt32(Console.ReadLine()) Переобразует введениые данные в числа
